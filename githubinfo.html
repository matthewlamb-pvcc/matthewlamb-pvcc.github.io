All About GitHub

Question Answer
What is a version control system? Software tools that help software teams manage changes to code over time
What is Git? A version control system used for tracking changes in computer files
What is GitHub? Code hosting platform for version control and collaboration
What is the difference between Git and GitHub? Git is a version control system that allows developers to track changes in their code. GitHub is a web-based hosting service for git repositories
Who started Github and how was it started? Chris Wanstrath, P.J. Hyett, Tom Preston-Werner, and Scott Chacon. Started with Ruby on Rails
What company owns it now? Microsoft Corp
How much does a Github account cost? Free but can be upgraded to a paid plan
What is the Octocat? GitHub's mascot. An anthropomorphized "octocat" with five octopus-like arms
Terms Definitions
Repository Place to store code, files, and each file's revision history
Commit Changes made to a file
Fork A new repository that shares code and visibility settings with the original "upstream" repository
Push Used to upload local repository content to a remote repository
Pull Requests Lets you tell others about changes you've pushed to a branch in a repository on GitHub
Workflows A configurable automated process that will run one or more jobs
Issues Let you track your work on Github
Raw Button Opens the file in a raw form, meaning that any HTML formatting disappears
Blame Button Gives you a line-by-line revision history, with the code in a file separated by commit
