Operating Systems Terminology
Term Definition
Operating System The software that controls the overall operation of a computer
Job Queue A storage organization in which jobs are ordered in first-in, first-out fashion
FIFO Objects are removed from the queue in the order in which they arrived
Interactive processing Remote terminals for having dialogue with a program being used
time-sharing Operating systems that provide service to multiple users
Multiprogramming Time is divided into intervals and then the execution of each job is restricted to only one intverval at a time
Application Software Programs for performing tasks particular to the machine's utilization
System Software Performs tasks that are common to computer systems in general
Utility Software Software that consists of software units that extend the capabilities of the operating system
Graphical User Interface (GUI) Objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
Shell Communicates with users through textual messages using a keyboard and monitory screen
Kernel Software components that perform the very basic functions required by the computer installation
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
File manager Coordinates the use of the machine's mass storage facilities
Memory manager Coordinates the machine's use of main memory
Virtual Memory "fictional" memory space created by paging
Boot strapping Procedure that transfers the operating system from mass storage into main memory
ROM Memory that can be read but not altered
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling Holding data for output at a later but more convenient time
Super user (or administrator) Has highly priviledged access to the operating system
Multi-factor authentication Requiring more than one piece of evidence to gain access
