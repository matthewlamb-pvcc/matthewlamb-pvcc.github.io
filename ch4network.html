Chapter 4 Terms
Network Classifications
    LAN: Local Area Network
    MAN: Metropolitan Area Network
    WAN: Wide Area Network
Network Models
    Bus Network: Machines communicate directly with each other over a common bus.
    Star Network: Communication is indirectly through an intermediary central machine.
Protocols
    CSMA/CD Ethernet: This protocol dictates that each message be broadcast to all of the machines on the bus.
    CSMA/CA Wifi: Protocols designed to avoid collisions but not eliminate them completely.
Combining Networks
    Repeater: Device that passes signals back and forth between the two original buses without considering the meaning of the signals.
    Bridge: Like a repeater but it does not necessarily pass all messages across the connection.
    Switch: Essentially a bridge with multiple connections allowing it to connect several buses rather than just two.
    Router: Special purpose computers used for forwarding messages.
    Gateway: The "point" at which one network is linked to an internet.
Client/server model
    P2P(peer to peer): Involves processes that provide service to and receive service from each other.
    Distributed system: Consists of software units that execute as processes on different computers.
Internet Addressing
    Tier 1: ISPs that consist of very high-speed, high-capacity, international WANs.
    Tier 2: ISPs that tend to be more regional in scope and less potent in their capabilities.
    Tier 3: An independent internet, sometimes called an intranet.
IP Address
    ICANN: A nonprofit corporation established to coordinate the Internet's operation.
    https://icann.org/
    DNS: Internet-wide directory system known as the domain name system.
    TLD (Top-level-domain): Domain suffixes which classify the domain.
    Country-code TLDS: Country specific domain suffixes.
    https://www.worldstandards.eu/other/tlds/
Internet protocols
    FTP:Application for listing and copying files across the network.
    HTTP: Webpages or Hyper Text Transfer Protocol.
    SMTP: Simple Mail Transfer Protocol. A way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
    URL: Unique web address for documents called Uniform Resource Locator.
    Client-side activities: Performed by a clienet(such as a browser)
    Server-side activities: Performed by a server(such as a webserver)
Internet four-layer software
    Application layer: Consists of software units such as clients and servers that use Internet communication to carry out their tasks.
    Transport layer: Accepts messages from the application layer to ensure the messages are properly formatted for transmission.
    Network layer: Decides in which direction a packet should be sent at each step along the packet's path through the Internet.
    Link layer: Transfers the packet and deals with the communication details.
Internet is
    Packet-switching network: Digital data is sent by breaking the data into blocks of bits called packets.
    Fault tolerant: Can fall back to other redundant pathways for delivery when congestion or failure occur.
    Redundant: More than one way to route data.
Cybersecurity
    Malware: Malicious software.
    Virus: Software than infects a computer by inseratin itself into programs that already reside in the machine.
    Worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
    Trojan-horse: A program that enters a computer system disguised as a desirable program, such as a game or useful utility package.
    Spyware: Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
    Phishing: A technique of obtaining information explicitly by simply asking for it.
    DDOS attack: When the work of generatin the attacks is distributed over many hosts.
Protection
    Firewall: A filter for traffic passing through a point in the network.
    Spam filter: Firewalls designed to block unwanted email.
    Proxy Server: A software unit that acts as an intermediary between a client and a server with the goal of shielding the client form adverse actions of the server.
    Antivirus software: Used to detect and remove the presence of known viruses and other infections.
    Cryptography: The study of sending and receiving messages securely in the presence of adversaries.
    Public key encryption: Techniques by which the encryption and decryption algorithms are known, but the encoded message remains secure.
