Terms

Definitions
Machine Language Instructions that are encoded as numeric digits
Debugging Correcting errors in a program
Assembler Program that converts mnemonic expression into machine language instructions
Assembly Language A mnemonic system for representing programs
Translator Translates programs expressed in high-level primitives into machine language programs
Compiler Compiles several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
Interpreter Executes a program from its high-level form
Cross-platform software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network
Natural Language Defined by grammars
Formal Language Evolved over time without formal grammatical analysis
Machine Independence Does not rely on the characteristics of a particular machine
High-level Programming Lanuage Provides more abstractions for the programmer and makes it easier for people to read and write a program
Object-oriented Programming A collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects
Scripting Language Used to perform administrative tasks rather than to develop complex programs
Data types:
Integer, Float, String		 Encompasses both the manner in which the data item is encoded and the operations that can be performed on that data
Primitive Data Type The data types that are included as primitives in a programming language, such as int for integer and char for character
Data Structure The conceptual shape or arrangement of data

Function A set of instructions for performing a task that can be used as an abstract tool by other program units
Local variable A variable declared within a function
Global Variable Variables whose scopes are not restricted to a particular part of a program
Source Program The program in its original form
Object Program The translated version of a program
